This is the documentation for my project 2. It will go over the brainstorming, coding, and optimizing of the project.

Brainstorming

I started by determining which type of API to use, and from the provided API's one stuck out to me. It was PokeAPI, which provided many data for various Pokémon. I decided to use this API to build a Pokédex like website. The main feature I wanted to implement was a search bar that would allow the user to search for a Pokémon by name or ID which would then display the Pokémon's information. Everything else was just extra features that I wanted to add to make the website more interesting. Once I got a few mockups created I started to code the website. It all started with a search bar and a div that would hold every information to be displayed. After creating the div, i went straight to JavaScript to start coding the search bar.
