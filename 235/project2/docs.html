Using The PokeAPI

What is the PokeAPI?

The PokeAPI is a RESTful API that provides data on the various Pokemon in the Pokemon franchise. It is a free API that is open to the public and is maintained by Paul Hallett. The API is updated regularly to include new Pokemon and new data on existing Pokemon.

The Main Features

  • The ability to login/register an account
  • Save your favorite Pokemon
  • Get information within a click
  • Randomly displays up to 32 Pokemon

How to Use

Using this website is simple, with the main feature being an infographic of a Pokemon of your choice. Simply type in the name or Pokedex id of the Pokemon you want to see and click the search button or press enter. You can also click the randomizer button to get a random Pokemon. Once you have a Pokemon, you can favorite it by clicking the favorite button while logged into an account.
