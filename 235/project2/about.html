WHY DID I MAKE IT?

"I want to be the very best. Like no one ever was." - Ash
I made this as a school project, but why did I choose Pokémon? Many people, including myself, grew up watching Pokémon. With various intricacies related to each Pokémon, this makes Pokémon appeal to people with diverse experiences. So, I wanted to create a website that lets anyone, regardless of experience, explore the various Pokémon as well as learn more about the ones you're interested in.

HOW DID I MAKE IT?

Here are the steps I used to create the best Pokédex website!

Step 1: Gathering intel and creating mockups of the site you envisioned.

Step 2: Soak in the API, relax and take your final breaths as you start consuming the API.

Step 3: Javascript time! Just jump straight into it, start by making all the neccessary API calls and get all the data!

Step 4: You got the data, you got the vision, now start building! One div at a time, start layering each info onto one another.

Step 5: Putting it all together, cook up the final divs and wa-la, you got yourself a fully functional site, without css.

Step 6: Cascading Style Sheets... alas the time has come, when all but css is done. Design principles, color theory, "why is this div not centering???", and rwd - responsive web design.

And that is how I created the best Pokédex website ever.
